# Proyecto Zamorano

Aplicación web jurídica profesional para administrar clientes, expedientes, agenda, pagos, documentos y escritos.

## Requisitos
- Node.js 18 o superior
- Visual Studio Code

## Instalar dependencias
```bash
npm install
```

## Ejecutar la aplicación
```bash
npm run dev
```

Luego abra la URL que indique Vite en el navegador.
